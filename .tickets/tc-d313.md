# Lint / checks in TypeScript port

Add quality checks and testing framework for the TypeScript port.

**Linting**: ESLint with TypeScript support
**Type checking**: TypeScript compiler (tsc)
**Testing framework**: Bun's built-in test runner

Add targets to the root `Makefile`:

```makefile
.PHONY: typescript typescript-lint typescript-type typescript-test typescript-check

typescript: typescript-check

typescript-check: typescript-lint typescript-type typescript-test

typescript-lint:
	@echo "Running TypeScript linter..."
	cd typescript/ticket && bun run lint

typescript-type:
	@echo "Running TypeScript type checker..."
	cd typescript/ticket && bun run typecheck

typescript-test:
	@echo "Running TypeScript tests..."
	cd typescript/ticket && bun test
```
