# Lint / checks in Zig port

Add quality checks and testing infrastructure for the Zig port.

**Linting/Format tools**:
- `zig fmt` - Zig's built-in formatter
- Run as part of checks to ensure code is properly formatted

**Type checking**:
- Zig has built-in compile-time type checking
- `zig build` will catch type errors

**Testing framework**:
- Use Zig's built-in test framework
- Add unit tests in source files
- Use `zig build test` to run tests

**Makefile targets** (add to root Makefile):

```makefile
.PHONY: zig zig-lint zig-type zig-test zig-check

zig: zig-check

zig-check: zig-lint zig-type zig-test

zig-lint:
	@echo "Running Zig formatter check..."
	cd zig/ticket && zig fmt --check src/

zig-type:
	@echo "Running Zig type checker..."
	cd zig/ticket && zig build

zig-test:
	@echo "Running Zig tests..."
	cd zig/ticket && zig build test
```

**Tasks**:
1. Add zig fmt check
2. Set up build configuration for tests
3. Add initial unit tests
4. Create Makefile targets
5. Verify all checks pass
